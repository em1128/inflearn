정규표현식

정규표현식 리터럴
var pattern = /a/;
a가 찾고자 하는 대상이라는 것을 컴퓨터에 알리고
pattern에 저장해 pattern으로 찾아냄.

정규표현식 객체 생성자
var pattern = new RegExp('a');
위와 같은 내용.
 : a를 찾기 위한 정규표현식

>>쓰임새
1. 필요한 정보 추출 : .exec
2. 필요한 정보가 있는지 확인 : .test
3. 특정 정보를 치환

정규표현식을 string과 연계해서 사용

.match : 문자열에 정규표현식이 있는지
.replace : 문자열에 정규표현식 패턴을 다른 것으로 치환

옵션i 
:정규표현식 리터럴에서 뒤에 i를 붙이면 대소문자 구분x
var oi =/a/i;
"Abcde".match(oi); // ["A"]

옵션g
:정규표현식 리터럴에서 뒤에 g를 붙이면 중복해서 찾아줌
var og = /a/g;
"abcdea".match(og); // ["a", "a"]

옵션 ig는 같이 사용가능

Regexper : 정규표현식을 시각화해서 보여주는 사이트

정규표현식에서 쓰이는 형식
\w : A~Z,a~z,0~9
+    : 앞에 있는 문자가 하나 이상일 때 사용
\s  : 공백

RegExr : 정규표현식 빌더 사이트
